# YUIOP29RE - DIY keyboard with rotaly encoder

## Requirements

* [PICO SDK][picosdk] (v2.1.1)
    * cmake
    * compiler or so
    * [picotool][picotool]
    

[picosdk]:https://github.com/raspberrypi/pico-sdk
[picotool]:https://github.com/raspberrypi/picotool

## How to build

```console
$ make
```
